markdown 学习笔记

markdown语法是一种标记语法。

一、标题

标题用“#”放在语句开头来标注，”#“个数= 标题等级。注意：”#“后面一般要加一个空格再编辑文字。

示例：

标题一

标题二

标题三

二、字体

加粗：

要加粗的文字左右两边用两个“*”

斜体：

在文字两边加一个”*“

斜体加粗：

文字两边加三个”*“

删除线：

文字两边加两个”~“

效果如下：

加粗 斜体 斜体加粗 删除线

三、引用

在引用的文字前加>即可。引用也可以嵌套。

示例：

引用

五、图片

语法：

！[图片alt]（图片地址”图片title“）

图片title：图片上显示的内容。
